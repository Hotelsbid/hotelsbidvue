# vps-manager
Демо-проект управления личным списком серверов VPS. 

## Цель
- Демонстрация подключения Supabase в проекты Vue
- Демонстрация возможностей Supabase

## Установка
```
npm install
```

### Локальная разработка (с горячей перезагрузкой)
```
npm run serve
```

### Подготовка к production
```
npm run build
```

### Проверка ошибок
```
npm run lint
```
